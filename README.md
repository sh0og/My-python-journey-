1.Backslash (`\`)
- Used to escape special characters in strings
- e.g:
print("It\'s Python")
Output:
It's Python

2.Backslash n (\n) 
-means a new line and is used inside a string. e.g:-
print("Hello\nWorld.") 
Output:
Hello
World.

3.end argument (end="") 
- The end argumwnt cntrols what is printed at the end of the output and keeps the next print on the same line.
-  e.g:
print("Hello Shoog",end="-")
print("Biscuit")
output:
Hello Shoog-Biscuit

4.sep argument (sep="")
- has an effect only when multiple arguments are provided to print().
- e.g:
print("Biscuit","shoog",sep="-")
Output:
Biscuit-shoog
